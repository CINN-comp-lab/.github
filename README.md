
# CINN Computational Lab

The **Computational Imaging and Neuroscience (CINN) Lab** at the University of Reading
develops open-source tools for neuroimaging research on the RACC2 HPC cluster.

## Packages

| Package | Description |
|---------|-------------|
| [CINN_fmriprep](https://github.com/CINN-comp-lab/CINN_fmriprep) | Run fMRIprep on the RACC2 HPC cluster via SLURM |
